# Social Preview Brief

Use this brief to create a GitHub social preview image for `openclaw-xhs-workflow`.

## Goal

Make it immediately clear that this repository is:

- an OpenClaw plugin
- for XiaoHongShu automation
- focused on draft-first content workflow

The image should help a developer understand the repo in under 3 seconds.

## Primary Message

`Turn one topic into a reviewable XiaoHongShu post pack`

## Secondary Message

`OpenClaw plugin for research, copy, image, review, and save-draft publishing`

## Short Variants

Use one of these if the layout is tight:

- `XiaoHongShu Workflow for OpenClaw`
- `Draft-First XiaoHongShu Automation`
- `OpenClaw Plugin for XiaoHongShu Posts`

## Visual Direction

- clean developer-tool aesthetic
- one strong headline
- one short support line
- avoid marketing-style gradients that feel generic
- use a workflow cue such as:
  - `research -> copy -> image -> review -> publisher`
  - or a pack folder / terminal / pipeline motif

## Content Blocks

Suggested top-to-bottom layout:

1. Brand line:
   `OpenClaw XiaoHongShu Workflow`
2. Headline:
   `Turn one topic into a reviewable post pack`
3. Support line:
   `research -> copy -> image -> review -> publisher`
4. Small footer:
   `draft-first publishing`

## Design Notes

- prefer dark text on light background or high-contrast neutral palette
- avoid overly cute social-media visuals
- target developers, builders, and AI workflow users
- the image should still look legible as a GitHub card thumbnail

## Image Model Prompt

If you generate this image with AI, start from this prompt:

`Design a GitHub social preview image for a developer tool called "OpenClaw XiaoHongShu Workflow". Clean technical aesthetic, modern editorial layout, high readability, neutral palette, subtle workflow diagram feeling. Main headline: "Turn one topic into a reviewable post pack". Secondary line: "research -> copy -> image -> review -> publisher". Small footer: "draft-first publishing". No clutter, no stock-photo style, no mobile-app marketing style.`
